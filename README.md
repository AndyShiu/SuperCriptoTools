# SuperCryptoTools
## 虛擬貨幣行情儀表板 (Beta版 v1.1)
* 本網頁工具提供使用者將自己關注的虛擬貨幣交易對的相關行情與價格在一個頁面上顯示。
* 本網頁工具所使用 **Window.localStorage** 網頁技術儲存個人設定資訊，該資訊儲存在您瀏覽電腦的本機上，並無存在任何網路或雲端空間。
